# Burner Book


**Burner Book is a privacy-focused Android application for managing temporary, self-destructing contacts. Built entirely with Jetpack Compose and modern Android technologies.**

Have you ever needed to share your contact information for a short-term engagement—like an online marketplace sale, a freelance gig, or a social event—but felt uneasy about them having your number forever? Burner Book is the solution.

This app allows you to create contacts with a built-in timer. Once the timer expires, the contact is automatically and permanently deleted from the app, keeping your primary address book clean and your privacy intact.

---

## ✨ Features

*   **Self-Destructing Contacts**: Set an expiry timer (from hours to days) for any contact. The app handles the cleanup automatically.
*   **Local-First Privacy**: All contacts are stored securely on your device's local database. No cloud sync, no data collection. Your privacy is paramount.
*   **Modern & Clean UI**: A beautiful, minimalist user interface built with Jetpack Compose and Material 3 design principles.
*   **Seamless Integration**: Make calls or send texts directly from the app using your phone's default dialer and messaging applications.
*   **Global Support**: An intuitive country code picker with flags makes it easy to add contacts from anywhere in the world.
*   **Data Management**: Simple options to export your temporary contacts for backup or import them when needed.
